### 階段 4：模型訓練（60 分鐘）

#### 4.1 訓練腳本
```python
from transformers import Trainer, TrainingArguments
from transformers import AutoTokenizer, AutoModelForSequenceClassification
import numpy as np
from sklearn.metrics import accuracy_score, f1_score, precision_score, recall_score

# 評估指標函數
def compute_metrics(eval_pred):
    predictions, labels = eval_pred
    predictions = np.argmax(predictions, axis=1)
    
    return {
        'accuracy': accuracy_score(labels, predictions),
        'f1': f1_score(labels, predictions, average='weighted'),
        'precision': precision_score(labels, predictions, average='weighted'),
        'recall': recall_score(labels, predictions, average='weighted')
    }

# 模型訓練配置
training_args = TrainingArguments(
    output_dir='./results',              # 模型和日誌保存目錄
    num_train_epochs=3,                  # 訓練 3 個 epoch
    per_device_train_batch_size=8,       # 訓練 batch size
    per_device_eval_batch_size=8,        # 驗證 batch size
    warmup_steps=500,                    # 預熱步數
    weight_decay=0.01,                   # L2 正則化
    learning_rate=2e-5,                  # 學習率
    logging_dir='./logs',                # 日誌目錄
    logging_steps=10,                    # 每 10 步記錄一次
    evaluation_strategy='epoch',         # 每個 epoch 評估一次
    save_strategy='epoch',               # 每個 epoch 保存一次
    load_best_model_at_end=True,         # 加載最好的模型
    metric_for_best_model='f1',          # 用 F1 選擇最好的模型
    greater_is_better=True,
    fp16=True,                           # 使用半精度加速（如果 GPU 支持）
)

# 初始化 Trainer
model = AutoModelForSequenceClassification.from_pretrained(
    "distilbert-base-uncased",
    num_labels=2
)

trainer = Trainer(
    model=model,
    args=training_args,
    train_dataset=train_dataset,
    eval_dataset=val_dataset,
    compute_metrics=compute_metrics,
)

# 開始訓練
print("開始訓練...")
trainer.train()
print("訓練完成！")

from transformers import pipeline

# 或手動推論
def predict_sentiment(text, model, tokenizer):
    """預測單條評論的情感"""
    inputs = tokenizer(
        text,
        max_length=256,
        padding='max_length',
        truncation=True,
        return_tensors='pt'
    )
    
    model.eval()
    with torch.no_grad():
        outputs = model(**inputs)
        logits = outputs.logits
        probs = torch.softmax(logits, dim=1)
        pred_label = torch.argmax(probs, dim=1).item()
        confidence = probs[0][pred_label].item()
    
    label_map = {0: 'Negative', 1: 'Positive'}
    return label_map[pred_label], confidence
```

## 🚨 常見錯誤 & 解決

| 錯誤 | 原因 | 解決方案 |
|------|------|--------|
| **CUDA out of memory** | Batch size 過大 | `BATCH_SIZE = 4`，或啟用 gradient_accumulation |
| **Token 數量不匹配** | Tokenizer 和加載模型不符 | 確保 tokenizer 和模型來自同一個 checkpoint |
| **Accuracy 不改進** | Learning rate 過小/過大 | 嘗試 1e-5 到 5e-5 範圍 |
| **FileNotFoundError** | 路徑錯誤 | 確保 train.csv、val.csv、test.csv 在工作目錄中 |
| **RuntimeError: grad can only be implicitly created for scalar outputs** | 損失函數異常 | 確認 num_labels=2 且標籤為 0/1 |
| **ImportError: accelerate>=1.1.0** | 缺少 accelerate 依賴 | `pip install accelerate>=1.1.0` |
| **GPU: False（無法使用 GPU）** | PyTorch 和 CUDA 版本不匹配 | 重新安裝 GPU 版本，或使用 CPU（會很慢） |
| **DistilBertForSequenceClassification LOAD REPORT: UNEXPECTED/MISSING** | 正常警告 | 可忽略，模型在改變任務時會重新初始化參數 |
| **數據已很乾淨，不需要清洗** | 不是錯誤 | 跳過 3.1 文本清洗步驟即可，直接進行標籤轉換 |
